# 🛍️ Projet Web E‑commerce – Jojgame

## 📌 Aperçu du projet

**Jojgame** est un site web e‑commerce développé dans le cadre d’un projet web, permettant d’exposer une **boutique en ligne** fonctionnelle.  
Le projet a été réalisé entièrement **sur IntelliJ IDEA**, ce qui a permis de gérer le code Kotlin/JavaScript/HTML/CSS dans un environnement professionnel intégré avec gestion de projet et Git.  

Le site permet aux utilisateurs de **parcourir des produits, consulter des détails, ajouter au panier et finaliser des achats**.  

🔗 Code source : [GitHub](https://github.com/Josue4231/projet-web-commerce-Jojgame-)

## 📁 Structure du projet
```css
/projet-web-commerce-Jojgame-
├── public/ # Fichiers public (HTML, CSS, images, JS)
├── src/ # Code source backend/frontend
│ ├── controllers/ # Contrôleurs (logique métier)
│ ├── models/ # Modèles de données / ORM
│ ├── routes/ # Définition des routes Web/API
│ └── views/ # Templates HTML ou vues
├── database/ # Scripts base de données / seeds
├── .gitignore
├── README.md # Documentation
├── package.json # Dépendances et scripts
└── server.js / app.js # Point d’entrée de l’application
```

💡 Projet développé et testé dans **IntelliJ IDEA**, facilitant la gestion du code et l’intégration avec GitHub.
